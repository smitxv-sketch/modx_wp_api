# AI Navigator

Добро пожаловать в проект! Это первый файл, который ты должен прочитать при начале работы над любой задачей. Он описывает общую архитектуру проекта, структуру директорий и расположение ключевых файлов.

## О проекте

Этот проект представляет собой дашборд (React + Vite + Express) для анализа и синхронизации данных из CMS MODX (в частности, для медицинских клиник: врачи, услуги, города). 
Бэкенд подключается к базе данных MySQL MODX для чтения данных (ресурсы, TV-параметры) и использует локальную базу данных SQLite (`sync_settings.db`) для хранения настроек (например, исключенные ресурсы).

## Структура проекта

- **`/src/`** — Исходный код фронтенда (React, TypeScript, Vite, Tailwind CSS).
  - `App.tsx` — Главный компонент приложения, содержит роутинг/вкладки (Overview, Doctors, Services, Graph, Config, SEO Analytics).
  - `components/` — UI компоненты (карточки, модалки, графики, отчеты).
  - `types/` — TypeScript интерфейсы и типы.
  - `index.css` — Глобальные стили (Tailwind).

- **`/server/`** — Исходный код бэкенда (Express, Node.js).
  - `server.ts` (в корне) — Точка входа для Express сервера.
  - `server/routes/api.ts` — API эндпоинты для работы с БД MODX и локальной SQLite.
  - `server/db.ts` — Настройка подключения к MySQL (MODX) и SQLite (локальные настройки).

- **`/scripts/`** — Скрипты для анализа БД MODX, тестирования запросов и исследования структуры (перенесены из корня для порядка).
  - `analyze-modx.ts`, `discover.ts`, `explore-db.ts` и др. — Полезны для понимания структуры таблиц и TV-параметров MODX.

- **`/docs/`** — Папка для документации проекта (если появится дополнительная документация, она будет храниться здесь).

- **Корневые файлы:**
  - `ai-navigator.md` — Этот файл (карта проекта).
  - `sync_settings.db` — Локальная SQLite база данных (не удалять, используется бэкендом).
  - `.env.example` — Пример переменных окружения (настройки подключения к БД).
  - `package.json` — Зависимости и скрипты запуска.
  - `vite.config.ts` — Конфигурация сборщика Vite.

## Технологический стек
- **Frontend:** React 19, TypeScript, Vite, Tailwind CSS, Lucide React, React Force Graph 2D (для графов), React-to-Print (для PDF отчетов).
- **Backend:** Node.js, Express, MySQL2 (для MODX), Better-SQLite3 (для локальных настроек).

## Как работать с проектом
1. Всегда начинай с чтения этого файла (`ai-navigator.md`).
2. Если нужно изменить UI — смотри в `/src/components/`.
3. Если нужно изменить логику работы с БД или добавить API эндпоинт — смотри в `/server/routes/api.ts` и `/server/db.ts`.
4. Для понимания структуры БД MODX можно заглянуть в скрипты в папке `/scripts/`.
5. При добавлении новой документации, помещай её в папку `/docs/`.
